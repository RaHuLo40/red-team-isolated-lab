# Red Team Cybersecurity Home Lab

This repository documents the design, setup, and isolation of a personal **cybersecurity home lab** built for **red team / offensive security practice**.

The lab is intentionally **fully isolated** to ensure safe exploitation of vulnerable systems without risking the host machine or external networks.

This project focuses on:
- Secure lab design
- Network isolation
- Ethical hacking environment setup
- Hands-on red team learning
